## Cell-level 재산출 — RAG ON 기준

기존 산출은 언어별 batch 호출 기준이므로 실제 integrated pipeline보다 낮게 잡힌다.  
아래는 실제 앱 구조에 맞춰 **셀 × 타겟 시트 단위 호출**로 다시 산출한 값이다.

- 번역 모델: `gemini-3.5-flash`
- 검수 모델: `gpt-5.4-mini`
- RAG: 항상 ON, 셀별 `n_results=2`
- 호출 단위: `source cell × target sheet`
- 환율: 1 USD = 1,501 KRW
- 산출 스크립트: `scripts/token_cost_report_cell_rag.py`

```bash
venv/bin/python scripts/token_cost_report_cell_rag.py \
  --audit-txt-006-a "tests/(CX Center) SmartThings_2.0_Story_Contents_006_0319_review_report_20260519_143205.txt" \
  --audit-txt-006-b "tests/(CX Center) SmartThings_2.0_Story_Contents_006_0319_review_report_20260519_144423.txt"
```
