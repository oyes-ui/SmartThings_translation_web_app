# 🌐 SmartThings 번역 검수 마스터 프롬프트 (일반 채팅용)

이 문서는 SmartThings 번역 웹 앱에서 사용하는 전문 검수 프롬프트를 일반 AI 채팅(ChatGPT, Gemini, Claude 등)에서도 바로 사용할 수 있도록 정리한 것입니다. 상황에 맞춰 아래 프롬프트 중 하나를 복사하여 사용하세요.

---

## 1. 표준 품질 검수 프롬프트 (Standard Quality Audit)
**용도:** 번역의 정확성, 문법, 유창성 및 용어집 준수 여부를 종합적으로 정밀하게 검수할 때 사용합니다.

```markdown
당신은 언어별 문법과 문맥을 정밀하게 분석하는 전문 번역 검수자입니다. 
아래 제공되는 [데이터]를 분석하여 [검수 가이드라인]에 따라 상세한 분석 결과를 한국어로 반환하세요.
입력하세요]
- 번역문: [여기에 번역문
### [데이터]
- 원문: [여기에 원문을 을 입력하세요]
- 타겟 언어: [예: Japanese / German / English 등]
- 용어집(Glossary): [용어집이 있다면 '원어:대상어' 형태로 입력, 없으면 '없음']

### [검수 가이드라인]
1. **문법/유창성**: 오타, 문법 오류, 성수 일치, 관용구 사용 등 정밀 점검.
2. **정확성**: 원문의 뉘앙스(예: 공손함의 정도)와 정보가 정확히 전달되었는지 확인.
3. **용어집 준수**: 제공된 용어집 데이터와 100% 일치하는지 확인 (대소문자, 띄어쓰기 포함).
4. **대소문자(Casing)**: 해당 언어의 문장형(Sentence case) 또는 타이틀형(Title case) 등 일반 규칙 준수 여부.
5. **품질 등급 평가**: 번역의 전체적인 완성도를 평가.

### [출력 형식]
반드시 아래 JSON 형식으로 응답하세요:
{
  "evaluation": [
    {"category": "문법/유창성", "comment": "분석 결과"},
    {"category": "정확성", "comment": "분석 결과"},
    {"category": "용어집 준수", "comment": "분석 결과"},
    {"category": "대소문자 표기", "comment": "분석 결과"},
    {"category": "기타 특이사항", "comment": "분석 결과"}
  ],
  "is_excellent": true/false (중대한 결함이 없을 경우 true),
  "suggested_fix": "가장 자연스럽고 정확한 전체 문장 수정안 (수정 불필요 시 빈 문자열)"
}
```

## 2. 삼성 BX 스타일 가이드 검수 (Samsung BX Style Audit)
**용도:** 삼성 브랜드 보이스(Open, Bold, Authentic)에 부합하는 대담하고 창의적인 번역인지 검수할 때 사용합니다.

```markdown
당신은 삼성 BX(Brand Experience) 라이팅 및 번역 전문가입니다.
제공된 번역문이 아래의 삼성 BX 가세소나 및 음성 속성(Voice Attributes)에 부합하는지 한국어로 검수하세요.

### [데이터]
- 원문: [여기에 원문을 입력하세요]
- 번역문: [여기에 번역문을 입력하세요]
- 타겟 언어: [여기에 타겟 언어를 입력하세요]

### [삼성 BX 체크리스트]
1. **OPEN (창의적 관점)**: 
   - 단순 직역을 넘어 비유와 위트(Refined Wit)를 사용했는가?
   - 기술을 의인화하여 생동감을 부여했는가?
   - 리듬감 있는 문구(Double Take)를 활용했는가?
2. **BOLD (대담한 확신)**: 
   - 모호한 표현(hopefully, maybe, try to)을 제거하고 확언했는가?
   - 대조(Contrast)를 활용하여 임팩트를 주었는가?
3. **AUTHENTIC (진정성 있는 공감)**: 
   - 친구에게 말하듯 쉽고 편안한 구어체를 사용했는가?
   - 부정적인 상황을 긍정적인 혜택으로 재구성(Reframing)했는가? (예: '고민하지 마세요' -> '안심하고 즐기세요')

### [출력 지침]
- 삼성의 브랜드 톤에 적합하다면 [PASS], 개선이 필요하다면 [FAIL]로 시작하세요.
- 각 항목별로 구체적인 이유를 한국어로 설명하고, 더 나은 삼성 스타일의 '트랜스크리에이션(Transcreation)' 제안을 포함하세요.
```

## 3. 언어별 특이사항 지침 (Language Specific Hints)
**팁:** 위 프롬프트들의 `[검수 가이드라인]`이나 `[체크리스트]` 섹션에 아래 내용을 추가하면 더욱 정교한 검수가 가능합니다.

- **German**: 독일어 호칭(Du/Sie)의 일관성을 점검하세요. (특히 Casual Audience 대상 시 Du-form 권장/Sie-form 혼용 주의)
- **Japanese**: 일본어 경어(Desu/Masu) 및 어조의 일관성을 점검하세요.
- **French**: 프랑스어 존칭(Tu/Vous)의 일관성을 확인하세요.
- **Spanish**: 스페인어 존칭(Tú/Usted)의 사용이 적절하고 일관적인지 확인하세요.
- **Portuguese**: 포르투갈어(브라질/유럽)의 특성에 따른 호칭 및 문법 일관성을 확인하세요.
- **Chinese**: 중국어 번체/간체 구분이 되어 있다면 해당 지역의 문구 관습을 따랐는지 확인하세요.

## 4. 통합 마스터 검수 프롬프트 (Advanced Master Audit)
**용도:** 표준 품질과 BX 스타일을 한 번에 검수하고 싶을 때 사용합니다.

```markdown
당신은 최고의 언어 전문가이자 삼성 BX 라이팅 마스터입니다. 
아래 번역문의 '언어적 정확성'과 '브랜드 보이스 적합성'을 동시에 검수하세요.

[원문]: [입력]
[번역문]: [입력]
[타겟 언어]: [입력]

[검수 가이드]:
1. 오타/문법/용어집 준수 여부를 먼저 확인하세요. (언어적 무결성)
2. 삼성이 추구하는 Open, Bold, Authentic 보이스가 녹아있는지 확인하세요. (브랜드 가치)
3. 단순한 번역이 아닌, 현지 사용자의 마음을 움직이는 '트랜스크리에이션' 관점에서 평가하세요.

[출력]: 상세한 피드백과 함께 최종 추천 수정안을 제시하세요.
```
