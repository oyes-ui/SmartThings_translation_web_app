# Slash Commands

이 폴더는 SmartThings translation skill용 slash command prompt 정의를 담는다.

도구별 slash command 설치 위치가 다를 수 있으므로, 이 패키지는 공통 원본만 제공한다. 설치 시 각 도구가 요구하는 commands 폴더로 필요한 파일을 복사한다.
