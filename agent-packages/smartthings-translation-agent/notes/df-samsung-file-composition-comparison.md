# 날짜 불일치 파일 확인 - DF vs 삼성 보유 구성 비교

## 기준

| 구분 | 의미 | 원본 기준 |
|---|---|---|
| 전자 [DF] | 처음 제공된 2개 사진을 1개 세트로 본 보유 목록 | 파일명에 날짜가 붙은 목록 |
| 후자 [삼성] | 이후 제공된 Finder 사진 기준 보유 목록 | Finder에 표시된 현재 보유 목록 |

> 이 문서는 파일명 날짜와 Finder 수정일 차이는 제외하고, 우선 "어떤 스토리/언어 파일을 보유하고 있는가"만 비교한다.

## 한 줄 결론

| 항목 | 결론 |
|---|---|
| 글로벌 구성 차이 | [DF]에는 있었으나 [삼성]에는 `025`, `039`, `051` 글로벌 파일이 보이지 않음 |
| CN 구성 차이 | [DF]에는 있었으나 [삼성]에는 `001_CN`, `004_CN`, `005_CN`, `006_CN`, `038_CN`이 보이지 않음 |
| 신규/대체로 보이는 다국어 파일 | [삼성] 쪽에 `006_IN`, `051_KR`이 새로 보임 |
| 가장 눈에 띄는 변화 | `006`은 CN에서 IN으로, `051`은 글로벌에서 KR만 보이는 형태로 관리 대상이 바뀐 듯함 |

## 전체 구성 차이 요약

| 구분 | [DF]에만 있음 | [삼성]에만 있음 | 메모 |
|---|---|---|---|
| 글로벌 파일 | `025`, `039`, `051` | 없음 | [삼성] Finder 목록에서는 3개 글로벌 파일이 빠진 상태로 보임 |
| CN 파일 | `001_CN`, `004_CN`, `005_CN`, `006_CN`, `038_CN` | 없음 | [삼성] 쪽에서는 해당 CN 파일 5개가 보이지 않음 |
| CN 외 언어 파일 | 없음 | `006_IN`, `051_KR` | `013_KR`, `044_IN`은 이전 세트에도 있었던 것으로 정리됨 |

## CN 외 언어 파일 비교

| 파일 | [DF] | [삼성] | 구성 판정 | 메모 |
|---|---|---|---|---|
| 006_IN | 없음 | 있음 | [삼성]에만 있음 | `006_CN`이 사라지고 `006_IN`이 보이는 형태 |
| 013_KR | 있음 | 있음 | 공통 | CN은 없고 KR만 있는 특이 케이스로 정리됨 |
| 044_IN | 있음 | 있음 | 공통 | `044_CN`과 별도로 IN도 존재 |
| 051_KR | 없음 | 있음 | [삼성]에만 있음 | [DF]의 글로벌 `051` 대신 [삼성]에는 KR만 보이는 형태 |

## 변경 포인트

| 포인트 | [DF] | [삼성] | 해석 |
|---|---|---|---|
| 006 | `006_CN` 존재 | `006_CN` 없음, `006_IN` 존재 | 중국 파일이 인도 파일로 대체되었거나 관리 대상 언어가 바뀐 것으로 보임 |
| 051 | 글로벌 `051` 존재 | 글로벌 `051` 없음, `051_KR` 존재 | 글로벌 파일은 현재 목록에서 빠지고 KR 파일만 별도 관리되는 형태로 보임 |
| CN 일부 파일 | `001_CN`, `004_CN`, `005_CN`, `006_CN`, `038_CN` 존재 | 해당 CN 파일 없음 | 삼성 보유 목록에서 CN 대상이 일부 정리된 것으로 보임 |

## 작업 메모

| 확인할 것 | 이유 |
|---|---|
| `025`, `039`, `051` 글로벌 파일이 [삼성]에 실제로 없는지 | Finder 캡처 범위에서 누락되었을 가능성 배제 필요 |
| `001_CN`, `004_CN`, `005_CN`, `006_CN`, `038_CN`이 [삼성]에서 삭제/제외된 것이 맞는지 | CN 번역 대상 제외인지, 파일 누락인지 구분 필요 |
| `006_IN`, `051_KR`의 출처 | 기존 CN/글로벌 파일 대체인지, 별도 신규 요청인지 확인 필요 |
